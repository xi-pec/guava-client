# Guava
GUAVA (General Utility Automated Vision Transformer Arranger) is a general item classifier
powered by OpenAI CLIP Vision Transformers. It comprises of two parts, [the client](https://github.com/xi-pec/guava-client)
and [the server](https://github.com/xi-pec/guava-server).

# How To Use
0. Install dependencies (i.e. `npm install`)
1. Run (i.e. `npm run dev`)

# Technologies Used
- Typescript
- Vite
- React
- Tailwind
- HeroUI
